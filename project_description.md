# Project Description

This project is a Restaurant Application.

Users can:

- Browse restaurants.
- Search restaurants.
- Filter restaurants.
- View restaurant details.
- Add restaurants to favorites.
- Make reservations.
- Manage their profile.

The application is built using Flutter, GetX, and MVC architecture.

The main goal is to provide a clean, responsive, and maintainable application with reusable components and no hard coded values.
